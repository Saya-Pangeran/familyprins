WELCOME TO OUR ASTRONOMY PAGE

 

 

 

 

We felt the urge to calculate the curve a space object is going to make when it enters the field of gravity of a planet. Next we would like to make this curve visible on the screen (See above some screenshots of the program. Click to enlarge pictures).
For this purpose a program was developed and made available for the visitor of this page. The program is written based on Visual Basic 2005 and can be downloaded below.

For the application of the program the following prerequisites are required: .NET Framework 2.0. If these components are already installed, you can launch the application now. Otherwise, click here to install the prerequisites and run the application. If you prefer, the prerequisites can also be downloaded directly from the Microsoft site: Microsoft downloadsite

The underlaying calculation is shown by clicking on 'Theory'.
We also studied the Special Relativity Theory of Albert Einstein. Important items of this theory are elucidated: the derivation of the Lorentz Transformation, Time Dilation, Lorentz Fitzgerald contraction and the relativity formula for adding two velocities. Enclosed is also the workout of the relativity paradox of the Barn and the Pole.
